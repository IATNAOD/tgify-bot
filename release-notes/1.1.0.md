# 1.1.0

## API Updates

- Support for [API 7.11](https://core.telegram.org/bots/api-changelog#october-31-2024).
- All methods and update types up to this API version are now fully supported.

## Added the previously missing button

- You can now create a button that copies text to the userâ€™s clipboard:

  ```TS
  const { Tgify, Markup } = require("@tgify/bot");

  const BOT_TOKEN = "xxx:xxx";

  const bot = new Tgify(BOT_TOKEN);

  bot.start((ctx) =>
  	ctx.reply("Start", {
  		reply_markup: Markup.inlineKeyboard([Markup.button.copyText("Hello", "super secret password", false, { style: "danger" })])
  			.reply_markup,
  	}),
  );

  bot.launch();

  // Enable graceful stop
  process.once("SIGINT", () => bot.stop("SIGINT"));
  process.once("SIGTERM", () => bot.stop("SIGTERM"));
  ```

## Added support for features related to paid content

- You can now create links with a monthly subscription to your channel:

  ```TS
  const { Tgify } = require("@tgify/bot");

  const BOT_TOKEN = "xxx:xxx";

  const bot = new Tgify(BOT_TOKEN);

  bot.start(async (ctx) => {
  	const subscriptionInviteLink = await ctx.telegram.createChatSubscriptionInviteLink("-xxx", 10);

  	await ctx.reply(`Invite: ${subscriptionInviteLink.invite_link}`);
  });

  bot.launch();

  // Enable graceful stop
  process.once("SIGINT", () => bot.stop("SIGINT"));
  process.once("SIGTERM", () => bot.stop("SIGTERM"));
  ```

- You can now send paid media content:

  ```TS
  const { Tgify } = require("@tgify/bot");

  const bot = new Tgify(BOT_TOKEN);

  bot.start(async (ctx) => {
  ```

- Additional features related to paid content:

  ```TS
  const { Tgify } = require("@tgify/bot");

  const BOT_TOKEN = "xxx:xxx";

  const bot = new Tgify(BOT_TOKEN);

  bot.start(async (ctx) => {
  	const transactions = await ctx.telegram.getStarTransactions();
  ```
