# 1.2.0

## API Updates

- Support for [API 9.4](https://core.telegram.org/bots/api-changelog#february-9-2026).
- All methods and update types up to this API version are now fully supported.

## Support for the following methods has also been added:

- `sendMessageDraft`, `getUserProfileAudios`, `getMyStarBalance`, `setMyProfilePhoto`, `removeMyProfilePhoto`, `giftPremiumSubscription`, `getUserGifts`, `getChatGifts` - new general bot feature

- `sendChecklist`, `approveSuggestedPost`, `declineSuggestedPost`, `editMessageChecklist`, `readBusinessMessage`, `deleteBusinessMessages`, `setBusinessAccountName`, `setBusinessAccountUsername`, `setBusinessAccountBio`, `setBusinessAccountProfilePhoto`, `removeBusinessAccountProfilePhoto`, `setBusinessAccountGiftSettings`, `getBusinessAccountStarBalance`, `transferBusinessAccountStars`, `getBusinessAccountGifts`, `convertGiftToStars`, `upgradeGift`, `transferGift`, `postStory`, `repostStory`, `editStory`, `deleteStory` - new bot features related to business accounts

<i>They are well documented. You can explore the functionality directly in the code or starting from [this update](https://core.telegram.org/bots/api-changelog#april-11-2025) onward</i>
